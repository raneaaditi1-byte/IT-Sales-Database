# 🗄️ IT Sales Database — MySQL Project

A complete MySQL database designed for an IT 
infrastructure company to manage clients, products, 
orders and sales performance tracking.

## 📊 Database Schema
- **clients** — client information and type
- **products** — Dell, HP, Lenovo product catalog  
- **orders** — purchase orders from clients
- **order_items** — individual items per order
- **sales_exec** — sales team and performance

## 🔍 Queries included
- Basic SELECT, WHERE, GROUP BY queries
- INNER JOIN and LEFT JOIN across multiple tables
- 3-table JOIN for complete order details
- Window functions — RANK(), ROW_NUMBER()
- Subqueries for complex filtering
- Stored procedures for automated monthly reporting

## 💡 Business insights generated
- Total revenue per client
- Sales executive performance vs target
- Monthly revenue trends
- Top selling products by brand
- Clients with no orders (LEFT JOIN)
- Low stock alerts

## 🛠️ Tools used
- MySQL 8.0
- MySQL Workbench
